# TR

Актуальность работы обусловлена необходимостью создания быстрых и точных систем раннего предупреждения о цунами. 
Традиционные методы, основанные на пороговых значениях часто дают ложные тревоги и требуют времени на анализ экспертом.
Нейронная сеть на TensorFlow способна в доли секунды оценить риск, анализируя комплекс параметров подводного землетрясения (магнитуда, глубина, ощущаемая интенсивность и др). 
Это позволяет автоматизировать первичную оценку, сократить время до предупреждения и минимизировать человеческий фактор, что критически важно для спасения жизней в прибрежных зонах.
